## 初期ロード時の認証判定

アプリ初期ロード時に FastAPI の `/api/me` を呼び出し、認証状態を判定します。

| `/api/me` の結果 | フロントエンドの挙動 |
| --- | --- |
| `200 OK` | ログイン済みとしてメイン画面を描画 |
| `401 Unauthorized` | ローカルログイン画面を表示 |
