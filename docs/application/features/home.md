# Home（サービス Hub）

| 項目 | 内容 |
| --- | --- |
| 機能 ID | home |
| 状態 | 未着手 |
| フェーズ | 1 |

## 概要

ログイン後のデフォルト画面です。家族向けサービスへの入口（Hub）として機能します。

特に指定がなければ `/home` に遷移します。

## 構成

Home は次の 3 ブロックで構成します。

```text
/home
├── サービス一覧（カード）     … 各サービスへ遷移
├── クイックアクセス          … 画面遷移なしで重要操作
└── ダッシュボード            … 各サービスの重要指標
```

## サービス一覧

| サービス | 説明 | 遷移先 | 状態 |
| --- | --- | --- | --- |
| 買い物リスト | 買い物のリスト管理 | `/shopping` | フェーズ 1 |
| （将来） | 追加サービス | TBD | 未決 |

カードにはサービス名、短い説明、アイコン（初期はテキストでも可）を表示します。

## クイックアクセス

**初期はコードで固定**します。後からユーザー別・管理者設定への拡張を検討します。

| ID | ラベル | 動作 | 対象サービス |
| --- | --- | --- | --- |
| QA-001 | 買い物リストを開く | `/shopping` へ遷移 | 買い物リスト |
| QA-002 | アイテムを追加 | モーダルでデフォルトリストへ追加 | 買い物リスト |
| QA-003 | 未完了件数を確認 | ダッシュボードの該当指標をハイライト | 買い物リスト |

QA-002 は [`shopping-list.md`](shopping-list.md) の「デフォルトリスト」に追加します。

## ダッシュボード

各サービスから重要指標を集約して表示します。初期実装:

### 買い物リスト

| 指標 | 説明 |
| --- | --- |
| 未完了アイテム数 | 対象グループの全 active リスト合計の未チェック件数 |
| デフォルトリスト名 | 現在のデフォルトリスト |
| デフォルトリスト未完了数 | デフォルトリストのみの件数 |

将来: 他サービス追加時にウィジェットを拡張します。

## API

フェーズ1では Home 専用 API を作りません。フロントエンドが `GET /api/v1/shopping/summary` を直接呼び出します。

| メソッド | パス | 概要 |
| --- | --- | --- |
| `GET` | `/api/v1/shopping/summary` | ダッシュボード用データ |

`/api/v1/home/summary` は将来、複数サービスを集約する際に検討します。

### GET /api/v1/shopping/summary（レスポンス例）

```json
{
  "group_id": "uuid",
  "total_open_items": 12,
  "default_list_id": "uuid",
  "default_list_name": "スーパー",
  "default_list_open_items": 5
}
```

詳細: [`shopping-list.md`](shopping-list.md)

## 権限

家族全員（`user`, `admin`）が同じ Home を表示します。データは家族共有です。
